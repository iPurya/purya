Telegram Advertising Bot

<h1>Install</h1>

```
git clone https://github.com/iPurya/purya.git
cd purya
chmod +x *
./bot install
```

<h1>Launch</h1>

```./bot [number]```
Put bot number instead number ...

<h1>Autolaunch</h1>

```./bot autolaunch```
